# FAQ

## Do I need Papyrus Client for every server?

No. Only servers with `anticheat.client-integrity.enabled: true` require the mod or launcher.

## Can I use Prism Launcher instead?

Yes, if you install the `papyrus-shield` mod jar for your version from [Download](/download). The full Papyrus Client launcher is recommended for automatic updates.

## Why is the integrity mod hidden?

So players cannot disable mod reporting while still using the official launcher build. Mod-only downloads are for advanced users who manage their own Fabric instances.

## Does this replace GrimAC or Papyrus server anticheat?

No. Client integrity complements [Papyrus server anticheat](https://docs.sushii.dev/papyrus/configuration/papyrus-options). Server-side checks still detect movement, reach, and mining patterns.

## Is Forge supported?

No. Papyrus Client is Fabric-only.
