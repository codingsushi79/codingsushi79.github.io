# Requirements

| Component | Requirement |
|-----------|-------------|
| OS | macOS 12+ or Windows 10+ |
| RAM | 4 GB minimum for launcher + game |
| Java | 21+ (1.21.x) or 17+ (1.20.x) — launcher uses system `java` |
| Account | Microsoft account with Minecraft Java Edition |
| Server | Papyrus server (optional; any Fabric server works without integrity enforcement) |
